How It's Made:

Tech used: HTML, CSS, JavaScript

I basically started with just making a basic HTML format of my resume before I started searching for designs that were far more stylish than my original resume that I could bring to life by taking advantage of all the wonderful options that vanilla CSS provides you with. It took me a while to find the type of design that I wanted to base my website resume on, but eventually I ran into a few different templates that I liked and began to make some much needed HTML changes first since I noticed that adding CSS to the current basic template that I had was not giving me the results that I wanted. After I went deeper into HTML, I was finally able to add lots of CSS to the template and kept doing so until I got the exact look that I wanted. I then proceeded to add a few JavaScript elements that would allow you to change the background image of the resume when clicked. I'm overall happy with how this project turned out and feel like I have a much better understanding of CSS than I did before finishing this project since I already had a decent understanding of how HTML works when it comes to using the right tags and JavaScript programming because of all of the object-oriented programming practice that I have done in the past. I also got to learn about some cool things along the way that I knew nothing about, such as how to use premade icons from sources like fontawesome.com.
